# 📜 Relations 冒险者关系指南

欢迎来到 **Relations** 的世界！这是一个致力于让每一位冒险者都能在旅途中建立深厚羁绊的系统。在这里，你不再是孤独的行者，而是可以与他人结为挚友、兄弟、闺蜜，甚至是相伴一生的伴侣。

---

## 🌟 核心概念 (Core Concepts)

在开始之前，请熟悉以下三个概念，它们将贯穿你的社交旅程：

1.  **关系 (Relation)**: 你与其他玩家之间建立的特殊连接。不同的关系类型（如伴侣、兄弟）拥有不同的权益和限制。
2.  **亲密度 (Affinity)**: 衡量你们关系深浅的数值。亲密度越高，你们的关系等级越高，解锁的特权（如专属称号、特效、礼物）也越丰富。
3.  **契约 (Request)**: 建立关系的正式邀请。只有双方都同意，关系才能成立。

---

## 🤝 第一章：建立羁绊 (Bonding)

### 1. 发起请求
你可以通过两种方式向其他玩家发送关系请求：

*   **快捷方式 (推荐)**:
    1.  按住 `Shift` 键。
    2.  对着目标玩家点击 `右键`。
    3.  在弹出的菜单中选择你想要建立的关系（如“求婚”、“结拜兄弟”）。

*   **咒语方式 (指令)**:
    输入以下指令发送请求：
    ```
    /rel request <玩家名> <类型>
    ```
    *例如：`/rel request Notch best_friend`（请求与 Notch 成为挚友）*

### 2. 接受/拒绝
当收到请求时，你的聊天栏会显示提示。你可以点击提示中的按钮，或者输入指令：
*   **接受**: `/rel accept <玩家名>`
*   **拒绝**: `/rel deny <玩家名>`

> **✅ 成功后**: 你们将正式确立关系，并开始记录亲密度。

---

## ❤️ 第二章：亲密度与升级 (Affinity)

亲密度是关系的灵魂。

### 1. 获取亲密度
*   **在线时长**: 与你的伙伴同时在线，亲密度会随时间自然增长。
*   **赠送礼物**: 使用特定的亲密度物品（如鲜花、巧克力）右键点击对方，可以大幅增加亲密度。
    *   *提示：不同的关系类型可能需要不同的礼物。*

### 2. 等级特权
随着亲密度的积累，你们的关系等级会提升，并解锁以下特权（视服务器配置而定）：
*   **Lv.1**: 基础关系确立。
*   **Lv.2+**: 解锁全服公告、特殊称号、入场特效等。
*   **更高等级**: 可能获得属性加成（如生命值提升、移动速度增加）。

## 💒 第三章：婚姻专属 (Marriage)

婚姻是最神圣的关系，拥有独一无二的特权。

### 1. 专属家园
你们可以拥有一个共同的家：
*   **设置家**: 站在你想定居的地方，输入 `/rel marry sethome`。
*   **回家**: 无论身在何处，输入 `/rel marry home` 即可瞬间传送回家。

### 2. 甜蜜互动
*   **亲吻**: 按住 `Shift` + `右键` 点击你的伴侣，可以进行亲吻互动（伴随爱心特效）。
*   **传送**: 输入 `/rel marry tp` 可以直接传送到伴侣身边（需对方在线）。
*   **互赠礼物**: 手持物品输入 `/rel marry gift`，可以将手中的物品直接塞进对方的背包。

## ⚔️ 第四章：关系管理 (Management)

### 1. 查看列表
想知道你有哪些关系？
```
/rel gui
```
*打开关系菜单，查看所有好友、亲密度等级和详细信息。*

或者使用指令：
```
/rel list
```

### 2. 查看排行榜
看看谁是服务器里最亲密的伙伴：
```
/rel top <类型>
```
*例如：`/rel top marriage` 查看婚姻亲密度排行。*

### 3. 设置性别
为了更好地展示身份，你可以设置自己的性别：
```
/rel gender <male|female|other>
```

### 4. 解除关系
缘分已尽？你可以单方面解除关系：
```
/rel remove <玩家名>
```
*警告：此操作不可逆，亲密度将清零。*

## ❓ 常见问题 (FAQ)

**Q: 我可以有多个伴侣吗？**
A: 不可以。婚姻关系通常是唯一的（取决于服务器配置，默认为 1）。但你可以拥有多个兄弟或挚友。

**Q: 为什么我不能发送请求？**
A: 可能是因为：
1. 你或对方的关系位已满。
2. 你们已经有关系了。
3. 对方开启了拒绝请求模式。

**Q: 亲密度有每日上限吗？**
A: 有的。为了保持平衡，每天通过道具获得的亲密度通常有上限。达到上限后，今日将无法继续通过道具增加亲密度。
